# Talkdesk's

> This application requires Node.js v8 + to work.

## CLI
```sh
$ npm install
$ npm link
$ tallkdesk-phone phones ./phones.txt
```

## Local
```sh
$ npm install
$ npm run start
```

SITE >> http://localhost:3001/
